#  Calculadora de Aportes para Dependientes

##  Introducción
En este repositorio comparto los archivos del proyecto.
esta se desarrollo en Visual Studio Code con HTML, CSS, 
JavaScript. Cuenta con programación básica para desarrollar funciones específicas como sacar 
porcentajes en las diferentes variables y por último mostrar el total de aportes. 
El framework con el cual se trabajo para la parte de Front-End fue Bootstrap una herramienta fácil 
de usar y de manera llamativa para ordenar la aplicación web con componentes, iconos, barras y 
colores.  
Se trabajo con CSS y JS lo cual fue de gran ayuda a la hora de mostrar ventanas e 
información. 
Por otra parte, en visual Studio se creo un index(codigo) el cual esta con comentarios que hace 
más fácil su lectura, cuenta con un orden y con una con una imagen corparativa que se 
utilizaron durante este proyecto.

##  Tecnologías Utilizadas
* **Lenguajes:** HTML, CSS y JavaScript (Lógica básica para porcentajes).
* **Framework Frontend:** Bootstrap (Utilizado para el orden de componentes, iconos y barras).
* **Editor:** Visual Studio Code.

##  Estructura del Proyecto
* `index.html`: Contiene el código fuente comentado para facilitar su lectura.
